# AI Face De-aging Web App

இது ஒரு புகைப்படத்தில் உள்ள முகத்தின் வயதைக் குறைத்து காட்டும் ஒரு AI செயலி.

## எப்படி இயக்குவது?
1. இந்த Repository-ஐ கிளான் (Clone) செய்யவும்.
2. `pip install -r requirements.txt` கட்டளையை இயக்கி லைப்ரரிகளை நிறுவவும்.
3. `streamlit run app.py` மூலம் செயலியைத் தொடங்கவும்.

## தொழில்நுட்பங்கள்
- Python
- Streamlit
- PyTorch (StyleGAN/SAM)
